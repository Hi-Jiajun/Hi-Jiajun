## :compass: How I Build / 做事方式

- Build useful things first. 先做有用的，再做漂亮的。
- Automate the boring parts. 优先自动化重复劳动。
- Make failures inspectable. 出错时要能看懂，也能定位。
- Prefer small tools that compose well. 我更喜欢可以自由组合的小工具。
- Be clear about ownership. 该说是自己做的就说自己做的，不是就明确写清楚。
